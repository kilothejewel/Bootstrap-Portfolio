# Bootstrap-Portfolio
Professional Portfolio - Cameron Darries
A sleek, responsive personal portfolio website built with Bootstrap 5.3. This project demonstrates clean UI/UX design, mobile-first responsiveness, and semantic HTML structure.

Key Features
Modern Hero Section: Full-screen landing page with a high-contrast dark overlay for maximum readability.

Dynamic About Card: A custom-designed card layout featuring technical skill badges and contact information.

Responsive Project Grid: A 6-project showcase utilizing a fluid grid system (1 column on mobile, 2 on tablet, 3 on desktop).

Uniform Card Logic: Used Flexbox mt-auto to ensure all action buttons align perfectly across different screen sizes.

Validated Contact Form: A clean, user-friendly form with built-in HTML5 validation and social media integration.

Built With
Bootstrap 5.3: For layout, grid, and pre-built components.

Bootstrap Icons: For professional vector iconography.

HTML5 & CSS3: For custom styling and structural integrity.
